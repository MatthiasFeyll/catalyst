# About Me - Personal Website

This is the third iteration of my personal website, built with **Next.js, React, and Tailwind CSS**. This project serves as an ongoing learning experience, where I explore different technologies by rewriting the site over time.

## Features

- **Modern Tech Stack**: Built with Next.js for SSR and React for dynamic components.
- **Tailwind CSS**: Ensures a responsive and minimalistic design.
- **Self-Improving Project**: Rewritten with different technologies to enhance skills.
- **AI-Assisted Development**: Built in just a few days with the help of AI, showcasing how AI can improve productivity and accelerate project development.

## Live Demo

[Visit my website](https://matthias-feyll.de)

## Installation & Setup

To run this project locally, follow these steps:

```sh
# Clone the repository
git clone https://gitlab.com/M-Felly/median_ai.git matthias_feyll_personal_page
cd matthias_feyll_personal_page

# Install dependencies
yarn install
yarn dev
```

The website is now available on [localhost:3000](http://127.0.0.1:3000)

## Deployment

This website is deployed on **Vercel** for fast and seamless updates
